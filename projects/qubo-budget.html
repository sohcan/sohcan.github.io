QUBO Budget Optimizer

A toy model for project selection under budget constraints

This project explores how Quadratic Unconstrained Binary Optimization (QUBO) can be used to model a practical planning problem: choosing the best set of candidate facility projects without exceeding a fixed budget.

The Problem

Real planning work often involves a list of possible projects, each with a cost, priority, risk impact, and timing constraint. The question is not simply “which project is best?” The harder question is:

Which combination of projects gives the strongest overall value while staying within the available budget?

The Toy Model

Each project is represented by a binary decision variable: 

xᵢ = 1 if project i is selected
xᵢ = 0 if project i is not selected

The model rewards high-value projects and penalizes project combinations that exceed the budget. 

minimize:  - project value + budget penalty

In plain English: pick valuable projects, but make going over budget expensive.

Why This Matters
